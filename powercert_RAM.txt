LA RAM, OU RANDOM ACCESS MEMORY

	La RAM, également appelée mémoire primaire ou stockage temporaire, constitue l'un des composants les plus importants des ordinateurs. Le terme RAM correspond à l'acronyme signifiant Random Acess Memory ou mémoire vive et est stockée sur la carte mètre dans des barrettes appelés DIMMs, pour Dual Inline Memory Module. Ils portent le nom de "Dual Inline Module" en référence au fait qu'ils présentent deux rangées indépendantes de pins, une de chaque côté. Une barrette de mémoire DIMM peut comporter 168, 184, 240 ou 288 pins. Pour être installée, la RAM est placée dans des slots de mémoire sur la carte mère, qui peut en comporter un nombre varié, mais généralement entre 2 et 4.

	Afin que des données ou un programme puissent fonctionner sur un ordinateur, il doit en premier lieu être chargé (loadé) dans la RAM. De ce fait, à la première étape les données sont stockées sue le disque dur, puis à partir de celui-ci, les informations sont chargées dans la RAM et dès lors, le CPU peut désormais accéder aux données en question ou faire tourner le programme.
	Dans beaucoup de cas, la mémoire est trop faire et peut potentiellement ne pas réussir à contenir toutes les données dont le CPU a besoin. Dans ce cas de figure, une partie des données doit être gardée dans le disque dur, plus lent, pour compenser ce manque. De ce fait, au lieu que les données aillent directement de la RAM au CPU, elles doivent transiter en temps réel entre le disque dur et la RAM, ce qui a pour conséquence de ralentir l'ordinateur. Pour résoudre ce problème, il suffit simplement d'augmenter la capacité de la RAM de la machine ce qui permettra donc de charger plus de données sans constamment nécessiter de demander la suite au HDD ; L'ordinateur sera de ce fait plus rapide. C'est pourquoi une machine avec plus de RAM fonctionnement plus rapidement qu'un ordinateur en possédant moins. Il faut tout de même noter que la RAM nécessite une mise sous tension constante pour fonctionner et stocker des données. Cela signifie donc que dans le cas où l'alimentation serait éteinte, les données seront effacées.
	*comme vous pouvez le constater, à cause de la place disponible dans la RAM, seulement les paquets de couleurs route, jaune, vert fondé, cyan et violet ont pu être chargés mais pas les paquets oranges, verts, et bleus. De ce fait, pour que le programme fonctionne correctement, il faut que la RAM récupère en deuxième temps ces paquets pour les faire parvenir au CPU et enfin faire fonctionner le programme.

	La RAM existe sous différentes formes ou types comme la DRAM, pour Dynamic RAM, qui a pour particularité de contenir des condensateurs. Ces derniers ressemblent grossièrement à des piles qui stockent de l'électricité. L'intérêt de leur présence sur les barrettes de DRAM est qu'ils permettent de retenir les bits d'information, en l'occurrence des 0 et des 1, si pendant un court instant la barrette venait à ne pas être alimentée. Etant donné que les barrettes de DRAM présentent des condensateurs, il faut tout de même les rafraichir en continu avec de l'électricité pour recharger ces derniers, puisqu'ils n'ont pas la capacité de fournir une tension sur une longue période. C'est ce rafraichissement en électricité qui donne le nom de Dynamic à la DRAM : les condensateurs doivent être dynamiquement rafraichis régulièrement, sans quoi la barrette de RAM oubliera les informations qu'elle contient.
	Un autre type de RAM que l'on peut citer est la SDRAM, ou Sychronous Dynamic RAM, utilisée aujourd'hui dans les DIMMs. Elles disposent comme pour les DRAM de condensateurs, et la différence entre les deux s'observe simplement dans les vitesses respectives. Les technologies les plus anciennes de DRAM fonctionnent de manière asynchrones avec l'horloge système, ce qui signifie qu'elles fonctionnent plus lentement que celles-ci étant donné que ses signaux ne sont pas synchronisés avec. En revanche, la SDRAM opère de manière synchrone avec l'horloge sytsème : c'est la raison pour laquelle elles fonctionnent plus rapidement que la DRAM. En effet, tous les signaux sont reliés à l'horloge interne de l'ordinateur de manière à obtenir un timing plus contrôlé. 

	Comme précisé précédemment, la RAM est installée sur la carte mère dans des DIMMS qui sont déclinés sous différentes tailles de mémoire. De nos jours, il est possible d'en trouver de capacités variant de 128Mo à 32Go. Les SDRAM sont également évaluées à différentes vitesses. Mais avant de pouvoir détailler cela, il faut définir certaines sous-notions : Le terme 32 ou 64-bit data path, ou chemin de donnée de bit, fait référence au nombre de bits de données qui sont transférés simultanément, ou en un cycle d'horloge. Plus le nombre de bits transféré en un cycle d'horloge est grand, plus la machine sera rapide. Les DIMMS possèdent un data path de 64 bits, ce qui signifie qu'ils peut transférer 64 bits de données en même temps. A titre de comparaison, les barrettes SIMMs antérieures aux DIMMs disposaient d'un data path de 32 bits, permettant donc de transférer 32 bits de données simultanément par cycle. La différence de bit data path constitue la raison pour laquelle les DIMMs sont plus rapides que les SIMM, puisqu'ils peuvent transporter deux fois plus de données pour un même cycle d'horloge.
	Un seul bit de données correspond à la plus petite forme de donnée qu'un ordinateur peut lire. En effet, une machine ne peut comprendre que des informations en base binaire, donc en 0 ou en 1, représenté par 1 seul bit sous deux états. On peut également distinguer le terme de byte ou octet, définit par la relation suivante : 1 byte = 8 bits. De ce fait, si un DIMM de mémoire est évalué à 64 bits de data path, cela signifie qu'il possède 64/8 = 8, soit un bus de 8 bytes de large.
	Les SDRAMS sont évaluées à plusieurs vitesses différentes. A titre d'exemple, les anciennes barrettes qui dataient des années 90 pouvaient être labellisées avec la mention "PC-100", pour laquelle le nombre 100 s'exprime en Hertz et détermine la vitesse à laquelle la RAM opère ; Puisque les SDRAM ne sont disponibles qu'avec des modules de 64 bits, elles possèdent des bus de 8 Bytes de large. A partir de données de fréquences en MHz et de largeur de bus, il est possible de calculer la bande passante de la barrette de SDRAM en procédant au calcul suivant : 100MHz * 8 bytes = 800MB/s ; On en déduit que la bande passante totale d'une SDRAM PC-100 est de 800MB/s. En d'autres termes, une barrette PC-100 peut transférer des données à une vitesse maximale de 800MB/s.
	Dans la même idée, une barrette de RAM labellisée avec le terme PC-133 signifie que sa vitesse d'opération ou fréquence d'opération est de 133MHz maximum ; Et dans le cas où elle possède des bus de 8 Bytes de large, la vitesse de transfert de données maximale sera de 133 MHz * 8 bytes = 1066 MB/s. La bande passante de PC-133 sera donc équivalente à 1066 MB/s. On pourrait au premier abord se dire que le résultat est rigoureusement faux, étant donné que 133 * 8 donne 1064, toutefois la vitesse de l'horloge indiquée à 133 MHz correspond en réalité à un arrondi de la valeur 133.3333 MHz, qui lorsqu'on le remplace dans le calcul nous donne bien le bon résultat.

	Un autre type de mémoire que l'on peut citer est la RDRAM, développée par Rambus inc, et qui ont conçu la RIMM dont l'acronyme signifie Rimbus Inline Memory Module. Celle-ci dispose de 184 pins et ressemble à la DIMM à l'exception que les encoches sont localisées au milieu du module. En 1999, la RIMM a été révolutionnaire en terme de mémoire mais a été rapidement dépassée à cause de la progression de la technologie des DIMMs. A savoir également que lorsque la RIMM faisait ses débuts, la vitesse de la RDRAM était de 800 MHz, ce qui était considérablement plus rapide que la SDRAM dont la vitesse était seulement de 133MHz environ à l'époque. Toutefois, bien qu'elle soit plus rapide que la SDRAM, la RDRAM ne possédait que des bus de 2 bytes de large, contrairement à la SDRAM qui en proposait des bus de 8 bytes de large. De ce fait, si l'on calcule la vitesse de la RDRAM, on obtient 800MHz * 2 bytes = 1600 MB/s.

	Comme la technologie s'est améliorée au film du temps et que les processeurs et les bus sont devenus plus rapides, une nouvelle technologie de RAM a fait son apparition pour suivre le rythme des ordinateurs. Celle-ci portait le nom de DDR, pour Double Data Rate, et comme son nom le laisse sous-entendre, permettait d'envoyer le double de la quantité de données pour chaque cycle d'horloges en comparaison avec la non DDR, ou Single Data Rate :
		<SCHEMA>
En effet, la non-DDR n'utilise que les paliers hauts du signal de l'horloge pour transférer des données, tandis que la DDR utilise à la fois les paliers hauts et bas. En conclusion, cela donne à la DDR la possibilité d'envoyer deux fois plus de données comparé à la SDR. Bien que la cadence d'horloge soit strictement identique, à temps égal, la DDR est donc bien plus efficace en manière d'envoi de données simplement en ayant l'avantage de se servir des deux états du signal de l'horloge. De ce fait, même si l'on augmente la fréquence de fonctionnement de celle-ci, la DDR ressortira toujours gagnant en terme de rapidité de transfert.
	La DDR et la non-DDR sont labellisées de manière différentes sur les barrettes de RAM puisque la DDR peut inclure à la fois la cadence de l'horloge et la largeur de la bande passante totale dans son nom. De ce fait, au lieu d'inclure seulement la vitesse de l'horloge dans l'intitulé, comme PC-133 où 133 équivaut à la cadence, la DIMM DDR inclus la largeur de la bande passante en plus. A titre d'exemple, une étiquette de barrette de RAM qui indique sur son étiquette DDR-333 PC-2700, le 333 indique la cadence de l'horloge et le 2700 la largeur de la bande passante totale réelle. Par ailleurs, on peut le confirmer par le calcul suivant :
			333 MHz * 8 Bytes = 2700 MB/s

	Une autre technologie ayant succédé à la DDR est la DDR-2. Cette dernière est plus rapide que sa version précédente puisqu'elle autorise les vitesses de bus supérieures et d'envoyer en l'effet deux fois plus de données que le DDR classique. Il nécessite de plus à la DDR-2 moins de courant que la DDR pour fonctionner. Additionnellement, la DDR-2 présente 240 pins contre 184 sur la DDR et sont toutefois labellisées d'une manière similaire, à la différence que les étiquettes de la DDR-2 disposent d'un '2' supplémentaire dans leur intitulé comme de la manière suivante : 
			DDR2-800MHz PC2-6400

	Successivement à la DDR-2, c'est au tour de la DDR-3 de faire son apparition. Comme pour le passage de la DDR à la DDR-2, la DDR 3 est deux fois plus rapide que la version précédente et nécessite également moins d'alimentation électrique que la DDR-2. Toutefois, bien que les DDR et DDR2 ne présentaient pas de différences au premier coup d'oeil, l'emplacement de l'encoche sur les barrettes DDR-3 est située plus à gauche en comparaison avec les deux générations précédentes. Il n'est donc pas possible d'installer une DDR-3 dans un slot de DDR ou de DDR2. En réalité, les cartes mères sont spécifiquement conçues pour accueillir un certain type de mémoire : il n'est par conséquent pas possible d'interconnecter des barrettes DDR, DDR-2 ou DDR-4 sur la même carte mère. Un exemple de label de DDR-3 pourrait être de la forme :
			DDR3-1600MHz PC3-12800

	La quatrième génération de DDR SDRAM est la DDR4 qui possède 288 pins contre 240 pour la DDR3. De la même manière que pour la génération précédente, il utilise également moins d'énergie pour fonctionner. La DDR-4 propose un plus grand éventail de vitesses comme la DDR4-4266MHz PC4-34100, qui possède une bande passante maximale de 34100 MB/s.

	Il existe des cas de figure pour lesquels la corruption de données mémoire ne peuvent pas être tolérées. A titre d'exemple les serveurs son conçus pour fonctionner en continu et certains serveurs ne peuvent pas se permettre d'être déconnectés pour une quelconque raison - comme les serveurs contrôlant des données de finances, des données médicales d'urgence, ou encore des données gouvernementales. Dans ce cas, les serveurs ne peuvent être déconnectés sous aucun prétexte, c'est pour cela que certaines barrettes de RAM possèdent des RCC embarquées, pour Error Connecting Code. La mission des ECC est de détecter si les données ont correctement été manipulées par la barrette mémoire et de corriger les erreurs de manipulation en cas de besoin.
	Pour déterminer	si une barrette de RAM présente un ECC ou non, il suffit de compter le nombre de puces sur le module : un DIMM Non-ECC possède 8 puces mémoire contre 9 sur les DIMM ECC. A noter que la majorité des barrettes mémoire de nos jours sont des Non-ECC. Il en est ainsi grâce aux avancements technologiques qui, avec le temps, ont permis de minimiser les erreurs mémoire et rendu les RAM Non-ECC plus stables. Le fait qu'elles soient moins courantes vient également du fait qu'elles soient majoritairement utilisées dans des serveurs, étant donné qu'ils doivent être fonctionnels continuellement. Dès lors, l'utilisation des barrettes RAM ECC constitue une précaution supplémentaire pour prévenir n'importe quelle erreur mémoire et protéger les serveurs contre les pertes de données mémoire.
